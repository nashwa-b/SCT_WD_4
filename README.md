A basic and elegant To-Do List web application designed to help users organize tasks efficiently. Users can create separate lists, add tasks with a description, set a date/time deadline, assign priority, and track completion.

---

##  Features

-  **Create Multiple Lists**  
  Choose between a default list or create a custom list with a unique name.

-  **Add Tasks**  
  Enter task descriptions along with deadlines in `dd-mm-yyyy --:--` format.

-  **Set Due Date & Time**  
  Each task can be associated with a specific date and time.

-  **Priority Levels**  
  Tasks can be categorized by priority: Low, Medium, or High.

-  **Mark as Completed**  
  Toggle tasks as done to keep your list clean.

-  **Edit and Delete Tasks**  
  Modify or remove tasks anytime.
